Documentación de Arquitectura 
Front-End: GEOLAND MVP (Premium 
Glassmorphism Edition) 
Objetivo: Construir la interfaz de usuario interactiva bajo el paradigma AI-First y 
Progressive Disclosure, utilizando un sistema de diseño "Titanium Glassmorphism" para 
proyectar exclusividad institucional y tecnología profunda ante inversores (VCs). 
 
1. Sistema de Diseño (Design System) 
La plataforma abandona el estilo tradicional de "portal inmobiliario" (fondos blancos, listas 
planas) para adoptar una estética inmersiva, translúcida y de alto contraste. 
●  Tema Base (Dark Mode Nativo): Fondos abstractos, texturizados o de gradientes 
muy sutiles en tonos oscuros (Grafito, Obsidiana, Midnight Blue). Cuando se analiza 
un activo (Capa 2), el fondo es la fotografía del activo en alta resolución a pantalla 
completa. 
●  Efecto "Frosted Glass" (Glassmorphism): Todos los contenedores de información 
son paneles translúcidos. 
○  Regla CSS Core: Uso de backdrop-filter: blur(16px) o superior, 
fondos semi-transparentes (rgba(255, 255, 255, 0.05)), y bordes 
ultradefinidos de 1px (border: 1px solid rgba(255, 255, 255, 
0.1)). 
●  Tipografía: Inter o SF Pro Display para limpieza absoluta. Tamaños de fuente 
masivos para las métricas core (G-Score, IRR) y pesos ligeros (Light/Regular) para 
el texto descriptivo. 
●  Paleta de Acentos: Uso restringido del color. Tonos oro pálido, esmeralda suave o 
azul eléctrico únicamente para el G-Score, las barras del radar ISV y los controles de 
los sliders. 
 
2. Stack Tecnológico Front-End 
●  Framework: Next.js (App Router, React 18+). Optimizado para precarga de 
imágenes de alta resolución y routing instantáneo. 
●  Estilos: Tailwind CSS. Se extenderá la configuración para incluir utilidades 
personalizadas de backdrop-blur y degradados de cristal.

●  Animaciones (El "Efecto Wow"): Framer Motion. Obligatorio para la expansión

4. Estructura de las Pantallas Principales 
Vista 1: AI-First Profiler (El Onboarding) 
●  Layout: Pantalla dividida asimétricamente. 
●  Lado Izquierdo (60%): AiChatProfiler. Una interfaz de chat limpia y sin bordes 
marcados. Las burbujas de la IA tienen el efecto cristal; las del usuario son más 
tenues. Aquí el Agente Perfilador extrae las 4 variables clave. 
●  Lado Derecho (40%): DynamicIsvRadar. Un panel flotante centrado que muestra 
el polígono del Investor Strategy Vector. Sus vértices se animan suavemente hacia 
los extremos a medida que el estado global de Zustand recibe el JSON oculto de la 
IA. 
----------------Page (1) Break----------------
Vista 2: Layer 1 - Executive View 
●  Layout: Grilla horizontal centrada. 
●  Componente Core: Layer1_AssetCard. 
●  Diseño Visual: Tarjetas de cristal. El número del G-Score ocupa un tercio de la 
tarjeta visualmente. Debajo, 3 métricas minimalistas (IRR, Liquidez, Riesgo). En la 
base, un párrafo de 2 líneas con el racional algorítmico generado por la IA. Todo flota 
sobre un fondo oscuro texturizado. 
Vista 3: Layer 2 - Analytical View (El "Aha Moment") 
●  Layout: Inmersivo total (Full Screen). Al hacer clic en una tarjeta de la Capa 1, la 
imagen de la propiedad se expande para ocupar todo el fondo de la pantalla. 
●  Componentes Core: Múltiples Layer2_MetricsPanel flotando como "widgets" 
sobre la foto de fondo. 
○  W idget Izquierdo: Gráfico de líneas (Escenarios Base, Optimista, Estrés). 
○  W idget Derecho Superior: Métricas específicas de la estrategia (ej. 
Cash-on-Cash, Yield on Cost). 
○  W idget Derecho Inferior: SensitivitySlider. Al arrastrar el control de 
"Costo de Reforma" o "Tasa de Salida", los números del widget superior y las 
líneas del gráfico izquierdo se actualizan instantáneamente a 60fps, 
calculados localmente en el front-end.
